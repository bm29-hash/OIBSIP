# Voice Assistant (Python Project)

This is a simple voice assistant built using Python. It can take voice commands and perform basic tasks like opening websites, telling time, and searching information.

## Features
- Voice recognition
- Text-to-speech response
- Opens websites (Google, YouTube)
- Tells current time
- Basic command execution

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- datetime
- webbrowser

## How to Run
1. Install required libraries:
   pip install speechrecognition pyttsx3 pyaudio

2. Run the program:
   python voice_assistant.py

## Example Commands
- "Open Google"
- "Open YouTube"
- "What is the time?"
