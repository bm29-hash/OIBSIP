# OIBSIP Internship Projects

This repository contains the projects I completed during my Python Programming Internship. These projects demonstrate my understanding of Python fundamentals, problem-solving skills, and practical implementation.

---

## 🔹 Projects

### 1. Voice Assistant

A Python-based voice assistant that can perform tasks using voice commands.

**Features:**

* Voice recognition
* Text-to-speech response
* Opens websites (Google, YouTube)
* Tells current time
* Executes basic commands

**Technologies Used:**

* Python
* SpeechRecognition
* pyttsx3
* datetime
* webbrowser

---

### 2. Password Generator

A simple Python program that generates secure and random passwords.

**Features:**

* Generates strong passwords
* Customizable length
* Uses letters, numbers, and special characters
* Enhances security

**Technologies Used:**

* Python
* random
* string

---

### 3. BMI Calculator

A Python program that calculates Body Mass Index (BMI) based on user input.

**Features:**

* Takes user input (weight & height)
* Calculates BMI
* Displays health category (Underweight, Normal, Overweight, Obese)
* Allows repeated calculations

**Technologies Used:**

* Python
